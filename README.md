# Task Manager App

A full-stack task management application built with the MERN stack (MongoDB, Express, React, Node.js).

## Features
- User signup and login with JWT authentication
- Each user can create, complete, and delete their own tasks
- Passwords are securely hashed using bcrypt
- Protected routes — only logged in users can access their tasks

## Tech Stack
- **Frontend:** React
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Authentication:** JWT, bcrypt
